# Analysis of Immigration Enforcement Data
This is a repo for data exploration and analysis of immigration enforcement 
data from the [Deportation Data Project](https://deportationdata.org/)[^1].

Currently, repo contains general cleaning and analysis of:
* Apprehensions
* Detention Stints
* Detainers

There is also a more detailed analysis of [Kansas City Apprehensions](https://rpubs.com/djknaggs/ice-arrests-kc).

## Caveats
While I have attempted to gain a reasonable understanding of the 
immigration system through the process of this work, I am absolutely not an expert. 
The purpose of this repository is to show one person's attempt at approaching some messy but 
crucially important datasets and provide the KCMO community with a data-focused 
overview of immigration enforcement actions in our area. Mistakes and oversights 
are my own and I appreciate any criticism on both technical approaches and data interpretations
(as well as ideas for additional analyses!).

Additionally, many of the scripts in this repo are works in progress. I will highlight
more polished analyses here as they come along.


[^1]: Hausman, David. 2025. “U.S. Immigration Enforcement Data.” Calif. L. Rev. Online 16 (13).
